プライバシーポリシー

最終更新日: 2026-01-18

Funny Reservation（以下「本サービス」）は、予約枠の確認および仮予約（Googleカレンダー予定の作成）を支援するためのAPIです。 本サービスは、利用者の入力内容をサーバー内DBに永続化せず、主に Google カレンダーを「スケジュールの正」として扱います。

1. 取得する情報

  • 予約枠確認: carId, startAt, durationMinutes
  • 仮予約作成: 上記に加えて customerName, carModelName, menuLabel, channel（任意）
  • 認証: Authorization ヘッダー（API_TOKEN）

2. 利用目的

  • 予約枠の空き状況の判定
  • 仮予約（Googleカレンダー予定）の作成による枠の占有
  • 不正アクセスの防止（API_TOKENによる認証）

3. 保存（永続化）について

本サービスは、予約情報をサーバー内のデータベース等へ永続保存しません。 ただし、仮予約作成時には Google カレンダーに予定が作成され、その内容は Google のシステム上に保存されます。

4. 第三者提供

本サービスは機能提供のために Google Calendar API を利用します。

5. セキュリティ

  • API_TOKEN による認証を行います。
  • 通信はHTTPSで行われます（Cloud Run）。
  • Googleへのアクセスはサービスアカウントにより行われます。

6. アクセスログ

Cloud Run / Google Cloud のログ（アクセス日時、リクエストメタ情報、エラー情報等）が記録される場合があります。 運用・障害対応・不正利用対策のために利用します。

7. お問い合わせ

本ポリシーに関するお問い合わせは、運用者の連絡先（LINE公式アカウント等）までご連絡ください。

8. 改定

必要に応じて変更することがあります。変更後は本ページにて公表します。
